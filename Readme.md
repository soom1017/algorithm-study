# 알고리즘 스터디
24년 겨울 알고리즘 스터디 저장소입니다.<br/><br/>
[백준](https://www.acmicpc.net/) 문제풀이 후 코드를 아래 형식으로 남기기 바랍니다.<br/>

```bash
├── 본인 이름
│   └── 주제
│       ├── [문제번호].py
│       ├── [문제번호].py
│       └── [문제번호].py
├── ...
└── README.md
```

일주일 최소 4문제 (골드 이상 3문제 포함) 풀이를 목표로 합니다.<br/>
문제 수 카운트는 커밋을 기준으로 하며, 이를 위해 커밋메시지를 `[_주차] 문제풀이`로 통일해주시면 감사하겠습니다.
